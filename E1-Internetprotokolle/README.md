# E.1 – Internetprotokolle

> **Dauer:** 5 Doppelstunden · **Werkzeug:** [Filius](https://www.lernsoftware-filius.de/) + Python (Thonny)

---

## Lernziele

Nach Abschluss dieses Themenfeldes könnt ihr:

- den Aufbau eines Rechnernetzes beschreiben und die Funktion von Komponenten (Switch, Router, …) erklären
- IP-Adressen und Subnetzmasken erklären und anwenden
- die Funktion des Domain Name Systems (DNS) beschreiben
- das TCP/IP-Referenzmodell mit seinen Schichten benennen und erläutern
- die Client-Server-Architektur erklären und Protokolle untersuchen
- Sicherheitsaspekte bei der Kommunikation im Netz benennen
- einfache Netzwerke in Filius aufbauen und testen
- eine einfache Netzwerkanwendung in Python schreiben
