# Vorlagen für das Abschlussprojekt

Diese Vorlagen dienen als Startpunkt für dein Abschlussprojekt (DS 13).

## Dateien

| Datei | Beschreibung | CodePen-Feld |
|-------|-------------|--------------|
| [projekt-vorlage.html](projekt-vorlage.html) | HTML-Struktur mit allen Abschnitten | **HTML** |
| [projekt-style.css](projekt-style.css) | Komplettes CSS mit Grid-Layout | **CSS** |
| [projekt-script.js](projekt-script.js) | JavaScript für Formular & Navigation | **JS** |

## So verwendest du die Vorlagen in CodePen

1. **Öffne** [codepen.io/pen](https://codepen.io/pen/)
2. **Kopiere** den Inhalt von `projekt-vorlage.html` → ins **HTML**-Feld
3. **Kopiere** den Inhalt von `projekt-style.css` → ins **CSS**-Feld
4. **Kopiere** den Inhalt von `projekt-script.js` → ins **JS**-Feld
5. **Fertig!** Du siehst sofort eine Vorschau

> **Hinweis:** In CodePen brauchst du kein `<!DOCTYPE html>`, `<html>`, `<head>` oder `<body>` – CodePen fügt das automatisch hinzu. Deshalb enthält die HTML-Vorlage nur den Inhalt, der normalerweise innerhalb von `<body>` steht.

## Anpassen

### Farben ändern
In der CSS-Datei findest du oben eine **Farbpalette**. Ändere einfach die Hex-Werte:
- `#2c3e50` → Hauptfarbe (Header, Footer, Navigation)
- `#3498db` → Akzentfarbe (Links, Überschriften-Unterstrich, Karten)
- `#34495e` → Sekundärfarbe (Navigation)

### Inhalte ändern
Ersetze die Platzhaltertexte im HTML durch deine eigenen Inhalte.

### Abschnitte hinzufügen/entfernen
Kopiere einen `<section>`-Block oder lösche einen, um die Seitenstruktur anzupassen. Vergiss nicht, auch den Navigations-Link anzupassen!
